Multivariable Analysis
Overview
This script performs multivariable Gaussian generalized linear regression analyses to identify factors associated with periodontal bacterial abundance.

The script generates:
Output	Description
Table S4	Multivariable regression coefficients (β) for individual bacterial species
Table S5	Fold-change estimates for red-complex abundance

The script uses the dataset generated by:
01_data_preparation.R

Outcomes Included in Table S4
Separate multivariable models were fitted for four periodontal bacterial species:
Repository variable	Species
log_p_gingivalis	Porphyromonas gingivalis
log_t_forsythia	Tannerella forsythia
log_t_denticola	Treponema denticola
log_p_intermedia	Prevotella intermedia

Bacterial abundance values were transformed using:
log10(x+1)

prior to modeling.

Explanatory Variables
All multivariable models included:
Maternal age
Gravidity
Passive smoking exposure
Alcohol use history
Drug use history
Dental visit history
Toothbrushing frequency
Toothbrush replacement interval

Categorical variables were included as factors using predefined reference categories established during data preparation.

Statistical Model
For each outcome, a Gaussian generalized linear model (GLM) was fitted:
Y=β0+β1X1+β2X2+⋯+βkXk

using:
glm(
  outcome ~ predictors,
  family = gaussian()
)
Complete-case analysis was applied.

Table S4
Table S4 reports:
Regression coefficients (β)
95% confidence intervals
p-values

for each explanatory variable in the four bacterial-species models.

β coefficients represent changes in log-transformed bacterial abundance associated with the explanatory variables.

Table S5
Red-complex abundance
Red-complex abundance was defined as:
Red Complex=P.gingivalis+T.forsythia+T.denticola

The summed abundance was transformed as:
log10(x+1)
and analyzed using the same multivariable model.

Fold-change calculation
To improve interpretability, coefficients were converted to fold-change values:
Fold Change=10β

The same transformation was applied to the confidence intervals.

Interpretation:
Fold change > 1 indicates increased abundance
Fold change < 1 indicates decreased abundance
Fold change = 1 indicates no difference
relative to the reference category.

Missing Data
All analyses were performed using complete-case datasets after removal of observations containing missing values in any model variable.
Missing-data summaries are reported separately in:
Table S3

Model Diagnostics
Assumptions of Gaussian GLMs were evaluated separately and reported in:
04_model_diagnostics.R

Diagnostic procedures included:
Shapiro–Wilk tests of residuals
Residual-versus-fitted plots
Normal Q–Q plots
Variance inflation factors (VIFs)
