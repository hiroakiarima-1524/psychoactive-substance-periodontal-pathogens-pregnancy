Descriptive Analyses
Overview
This script generates descriptive visualizations of participant characteristics and periodontal bacterial abundance.
Outputs produced by this script include:
Output	               Description
Figure 3	               Participant characteristics
Figure 4A	Distribution of bacterial abundance
Figure 4B	Correlation heatmap among bacterial species

The script uses the prepared dataset generated by:
01_data_preparation.R

Figure 3: Participant Characteristics
Figure 3 summarizes participant characteristics included in the study.
The following variables are visualized:
Panel	               Variable
Figure 3A	Maternal age
Figure 3B	Gravidity
Figure 3C	Toothbrushing frequency
Figure 3D	Toothbrush replacement interval
Figure 3E	Dental visit history
Figure 3F	Passive smoking exposure
Figure 3G	Smoking history
Figure 3H	Alcohol use history
Figure 3I	Drug use history

No inferential statistical testing was performed.
Histograms were used for continuous or ordinal variables, whereas bar plots were used for categorical variables.

Figure 4A: Distribution of Bacterial Abundance
The abundance distributions of four periodontal bacterial species were visualized using boxplots.
Included species:
Repository variable	Bacterial species
p_gingivalis	Porphyromonas gingivalis
t_forsythia	Tannerella forsythia
t_denticola	Treponema denticola
p_intermedia	Prevotella intermedia

Log-transformed abundance values generated in 01_data_preparation.R were used.

Figure 4B: Correlation Heatmap
Relationships among bacterial species were evaluated using Spearman's rank correlation coefficients.
The correlation matrix was calculated using complete observations only:
cor(
  bacterial_data,
  method = "spearman",
  use = "complete.obs"
)

The resulting matrix was visualized using the corrplot package.

Missing Data
Missing observations were excluded separately for each visualization.
