Model Diagnostics
Overview
This script evaluates the assumptions and diagnostic performance of multivariable Gaussian generalized linear models used to investigate associations between participant characteristics and periodontal bacterial abundance.

The script generates the following outputs:
Output	Description
Table S1	Shapiro–Wilk normality assessment of categorical variables
Table S2	Variance inflation factors (VIFs)
Table S3	Missing-data and complete-case summary
Reviewer response analysis	Residual diagnostics of Gaussian GLMs

The script uses the dataset generated by:
01_data_preparation.R

Table S1: Normality Assessment
To determine the suitability of parametric assumptions, Shapiro–Wilk tests were performed within each category of the explanatory variables.
Variables evaluated included:
Brushing frequency
Toothbrush replacement interval
Smoking history
Alcohol use history
Drug use history
Dental visit history
Passive smoking exposure

The outcome variable used for the assessment was periodontal bacterial abundance.

Table S2: Multicollinearity Assessment
Multicollinearity among explanatory variables was evaluated using variance inflation factors (VIFs).
The multivariable model included:
Maternal age
Gravidity
Passive smoking exposure
Alcohol use history
Drug use history
Dental visit history
Brushing frequency
Toothbrush replacement interval

VIF values below commonly accepted thresholds were considered indicative of low multicollinearity.

Table S3: Missing Data Assessment
Missing observations were summarized for all explanatory variables included in multivariable analyses.
Complete-case analysis was subsequently applied for regression modeling.

Residual Diagnostics
As requested during peer review, additional diagnostic procedures were performed to assess assumptions of Gaussian generalized linear models.
Diagnostic procedures included:
Shapiro–Wilk test of residual normality
Residual-versus-fitted plots
Normal Q–Q plots

Outcome Variables
Residual diagnostics were performed separately for the following periodontal bacterial species:
Porphyromonas gingivalis
Tannerella forsythia
Treponema denticola
Prevotella intermedia

Outcome Transformation
Because bacterial abundance data were highly right-skewed and included zero values, outcomes were transformed as:
log10(x+1)

before fitting Gaussian generalized linear models.

Software
Analyses were performed using:
R
dplyr
car

This script was used exclusively for model diagnostics and validation procedures and does not generate the primary regression results reported in the manuscript.
