Univariable Analyses
Overview
This script performs univariable analyses examining associations between participant characteristics and periodontal bacterial abundance.

The script generates:
Output	Description
Table 1	Group comparison of Tannerella forsythia abundance according to alcohol use history
Figure S1	Associations with Porphyromonas gingivalis abundance
Figure S2	Associations with Tannerella forsythia abundance
Figure S3	Associations with Treponema denticola abundance
Figure S4	Associations with Prevotella intermedia abundance

The script uses the dataset generated by:
01_data_preparation.R

Continuous Variables
Associations between bacterial abundance and continuous variables were evaluated using:
Maternal age
Gravidity

For each variable:
A scatter plot with a cubic polynomial smoothing curve was generated.
The maximal information coefficient (MIC) was calculated using the minerva package.

MIC values were displayed in the corresponding supplementary figures.

Categorical Variables
Associations between bacterial abundance and categorical variables were evaluated for:
Toothbrushing frequency
Toothbrush replacement interval
Dental visit history
Passive smoking exposure
Smoking history
Alcohol use history
Drug use history

Visualizations were generated using jittered scatter plots.
Statistical tests included:
Variable type	Statistical test
Two-group comparison	Wilcoxon rank-sum test
Three or more groups	Kruskal-Wallis test

P-values were displayed in the supplementary figures.

Normality Assessment
Shapiro-Wilk normality tests were performed separately and reported in:
Table S1
These tests are not included in this script.

Post-hoc Comparisons
Post-hoc Steel-Dwass tests were performed only for analyses reported in:
Table 1
and are therefore not included in this script.

Software
